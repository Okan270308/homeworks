Властивості блоків

Відступи padding та margin

Властивість padding — це внутрішній відступ. Використовується для відступів від кордону блоку всередину блоку (елементу), а не між блоками (елементами). Значення можуть бути лише додатними, тобто більше або рівне нулю.

Простими словами padding — це Відступи ВСЕРЕДИНІ об'єкту, від кордонів об'єкта ДО ЙОГО КОНТЕНТУ. НЕ успадковується.

При вказанні значення у відсотках, відсоток береться від ширини батьківського об'єкта, навіть для вертикальних відступів.

Властивість margin — Зовнішній відступ. Використовується для відступів між блоками (елементами), рекомендується робити відступи зверху вниз (margin-bottom) зліва на право. Значення можуть бути від'ємними! НЕ успадковується

Правило для відступів margin між елементами — відступи margin задаємо на одному рівні вкладеності.

Не допускається відступ від дочірнього елементу до наступного батьківського, тобто відступ з 2-го рівня вкладеності до першого

Схлопування margin (накладання одного margin на інший) — відбувається в кількох випадках:

1. Коли контент відсутній в блоках, тобто є два пустих сусідніх блоки з встановленими зовнішніми відступами.

2. Коли між двома сусідніми елементами заданий зовнішній відступ margin як це описано в прикладі вище.

Властивості для встановлення розмірів елементів

Властивість width - чітко задає ширину блочного (блочно-рядкового) об'єкту, НЕ працює з рядковими об'єктами

Bластивість max-width – встановлює **максимальну ширину** блочних (блочно-рядкових) тегів, тобто блок не буде більшим за вказану ширину. НЕ працює зі рядковими об'єктами. Не успадковується.

Властивість min-width — встановлює мінімальну ширину блочних (блочно-рядкових) тегів, тобто блок не буде меншим за вказану ширину. Не успадковується.НЕ працює зі рядковими об'єктами.

Властивість height — встановлює чітку висоту блочних (блочно-рядкових) тегів. Не успадковується.НЕ працює зі рядковими об'єктами

Ніколи не встановлюйте фіксовану висоту для контентних (текстових) елементів, тому що при зміні контенту (тексту) будуть проблеми.

Властивість min-height — встановлює мінімальну висоту блочних (блочно-рядкових) тегів, тобто блок не буде меншим за вказану висоту.Не успадковується.НЕ працює зі рядковими об'єктами

Властивість max-heightt — встановлює максимальну висоту блочних (блочно-рядкових) тегів, тобто блок не буде меншим за вказану висоту.Не успадковується.НЕ працює зі рядковими об'єктами

Ніколи не обмежуйте висоту для контентних (текстових) елементів, тому що при додаванні (збільшені кількості) контенту будуть проблеми
